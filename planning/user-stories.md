## user stories

* As a user, I want to view a bunch of blog post, so I can... read them.
* As a user, I want to view an individual blog post, so i can read it
* As a user, I want to be able to login / sign up, so I can comment on posts (?)
* As a user I want to be able to comment on an individual blog post, so I can troll
* As a user, I want to be able to post a blog post under my name, so I can share my genius with the world
